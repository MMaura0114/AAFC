Como usar o envio manual:

  1. Clique no botão "Gerar Lista de Números"
  2. Copie os números que aparecerão abaixo
  3. Abra o WhatsApp Web ou WhatsApp Desktop
  4. Clique em "Nova conversa" ou "Nova mensagem"
  5. Cole os números no campo de busca (um por vez)
  6. Envie a mensagem para cada contato

Obs: Para uma experiência melhor, recomenda-se usar no computador com WhatsApp Web/Desktop aberto.

Como funciona o envio automático:

  1. O sistema abrirá uma nova janela do WhatsApp Web para cada número
  2. A mensagem já virá pré-preenchida
  3. Você precisará clicar em "Enviar" manualmente em cada janela
  4. Haverá um intervalo de 5 segundos entre cada janela
  5. Mantenha o foco na janela do navegador durante o envio

Atenção: Esta funcionalidade pode não funcionar bem em todos os navegadores devido a bloqueios de pop-up.
