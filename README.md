# Sistema de Gerenciamento de Associados

## 📋 Descrição

Sistema web completo para gerenciamento de associações, desenvolvido em HTML, CSS e JavaScript puro. Permite o cadastro de associados, controle de contribuições mensais, envio de mensagens via WhatsApp e geração de relatórios financeiros.

## ✨ Funcionalidades

### 👥 Gestão de Associados
- **Cadastro de novos associados** com número automático
- **Consulta e edição completa** de dados dos associados
- **Controle de status** de pagamento mensal
- **Múltiplas formas de pagamento** (PIX, Dinheiro, Cartão, Transferência)

### 💰 Controle Financeiro
- **Registro de contribuições mensais**
- **Histórico completo** de pagamentos
- **Status visual** por mês (Pago/Pendente)
- **Cálculo automático** de meses em débito

### 📊 Relatórios e Análises
- **Dashboard administrativo** com métricas
- **Arrecadação por mês** e total
- **Relatórios por forma de pagamento**
- **Exportação de dados** em PDF e Excel

### 📱 Comunicação
- **Envios de lembretes** via WhatsApp
- **Mensagens personalizáveis** com templates
- **Programação de envios** para data futura
- **Lote para associados** com pendências

### 🔐 Sistema de Acesso
- **Login diferenciado** para administradores e associados
- **Área restrita** para cada tipo de usuário
- **Proteção de dados** no localStorage
